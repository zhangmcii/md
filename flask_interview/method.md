学习基础工具 
基础了解 
本文介绍如何使用 ChatGPT 提高学习基础工具的效率， 例如学习 Redis, Django, React, MySQL 等。
在面试中，常见的问题包括： 介绍此工具的使用场景，它的优点和缺点是什么？ 介绍此工具的关键特性以及实现原理 与其他相似的工具对比，什么情况下选择它？
确定关键主题 某些组件相对比较复杂，内容繁杂。例如数据库，消息队列等，所以一开始我们应该先确定学习的重点主题，使用合适的 ChatGPT prompts 非常重要，
这里以 Flask 为例，当然你可以替换成其他工具

# **📌 一、获取 Flask 面试重点主题**

用途：快速锁定应重点学习的领域。

### **Prompt**

```
你是一名高级后端工程师，请根据中国互联网公司常见面试内容，
列出“后端开发工程师（1-3 年经验）”在 Flask 方面最常被问的 7 个主题，
并按重要度从高到低排序。
要求给出主题名 + 简短解释（不超过 20 字）。
```

---

# **📌 二、针对某个主题生成 30 分钟学习计划**

用途：避免漫无目的阅读文档，提高效率。

### **Prompt**

```
请针对主题「Flask 路由与视图函数」，为我制定一个 30 分钟学习计划。
内容包括：
- 必读的官方文档章节（提供链接）
- 学习重点（3-6 条）
- 面试中需要会解释的问题（4 条）
- 我应该能够用 3 句话总结的内容
```

---

# **📌 三、在阅读文档前，让 GPT 做“阅读指引”**

用途：告诉你什么重要，什么可以跳过。

### **Prompt**

```
下面是 Flask 官方文档的 URL（例如：https://flask.palletsprojects.com/ ）。
请告诉我：为了学习主题「Flask 路由系统」，
文档中我必须重点阅读的部分有哪些？哪些部分可以跳过？请用列表形式回答。
```

---

# **📌 四、学习完后由 GPT 扮演面试官问你**

用途：模拟面试 + 及时反馈。

### **Prompt**

```
请扮演一个面试 Flask 的面试官，
针对主题「Flask 路由机制」向我提出 4 个由易到难的问题。
我回答后，请你：
1. 按 10 分制打分
2. 指出回答中不清晰、不准确的地方
3. 告诉我优秀答案应该是什么结构
```

---

# **📌 五、生成主题知识结构脑图（文本格式）**

用途：学习完一个模块后整理整体结构。

### **Prompt**

```
请为主题「Flask 框架核心机制」生成一份结构化脑图。
格式为：
- 一级主题
  - 二级主题
    - 关键知识点
    - 常见面试题
    - 常见误区
请尽量覆盖 Flask 的主流模块，输出为markdown格式
```

生成后你可直接复制到 XMind。

---

# **📌 六、生成应用练习（比面试题更有用）**

用途：让你真正“会使用”而非只会背答案。

### **Prompt**

```
请给我 3 个 Flask 的实战练习题，用来巩固主题「请求处理与响应」。
难度从简单到中等，每个练习包含：
- 任务描述
- 输入/输出示例
- 验收标准
- 常见错误
```

# **📌 七、在遇到不懂的概念时快速补课（3 分钟速通法）**

用途：避免被陌生概念卡住学习流程。

### **Prompt**

```
请用 3 分钟速通方式解释概念「Flask WSGI」：
- 用白话解释是什么
- 用一个例子说明它解决了什么问题
- 在 Flask 中它是如何运作的
- 面试中我需要记住的三点
```

复制后替换成任何概念都可以。
