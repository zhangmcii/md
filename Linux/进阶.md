## find
find 路径 条件 -exec 命令 {} \;
find 路径 条件 -exec 命令 {} +

解释：
-exec	告诉 find 要执行命令
命令	要执行的程序
{}	当前找到的文件
\;	命令结束符
+   命令结束符

\;（旧方式），每个文件执行一次， 会产生很多进程。
而 +（更高效），一次传多个文件，性能更好。


## sed : 文本替换工具
-i : 直接修改文件
s/old/new/g :  替换规则



4.这个挑战包含一些文本文件（扩展名为.txt），这些文件中都包含有“challenges are difficult”这一短语，请递归地从所有文本文件中删除这一短语。
注意，有些文件位于子目录中，因此您需要搜索它们。
```
find . -name "*.txt" -exec sed -i 's/challenges are difficult//g' {} +
```

## awk : 文本处理语言

5.文件sum-me.txt中包含一个数字列表，每行一个数字。请打印这些数字的总和。
```
awk '{sum += $1} END {print sum}' sum-me.txt
```

## find
-type f : 普通文件
-type d : 目录
-type l ：符号链接
-printf ：自定义输出格式
%f ： 仅文件名（不包含路径）
\n ：换行


6.递归打印当前目录下的所有文件，不包括前导目录路径。
```
find . -type f -printf "%f\n"
```

## mv : 移动和重命名文件


7.递归重命名当前目录中的所有文件，去掉它们的扩展名。
```
find . -type f -exec bash -c 'mv "$0" "${0%.*}"' {} \;
```

{}	: 当前文件
bash -c	 : 运行一段 shell 代码
"${0%.*}"  : 删除最后一个扩展名
