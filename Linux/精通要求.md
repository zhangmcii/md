# 1 最适合你当前水平的练习

## OverTheWire Bandit（强烈推荐）

这是个 Linux CTF 游戏。很多人用它从“会命令”练到“会思考”。

OverTheWire Bandit

特点：

* 通过 SSH 登录服务器
* 每一关要用 Linux 命令找密码
* 找到密码才能进入下一关

你会被逼着用：

```bash
grep
find
strings
sort
base64
chmod
ssh
```

示例题型：

```
在某个目录里找到唯一一个包含 "password" 的文件
```

你就必须组合：

```bash
grep -r password .
```

或者：

```bash
find . -type f -size 1033c
```

这种练习非常接近真实问题。

# 5 最实用的练习方式（很多人忽略）

说句经验之谈。最有效的练习其实是 **日志分析题**。

你可以随便找个 nginx 日志练：

```
access.log
```

然后练这些问题：

### 统计访问最多的 IP

```bash
awk '{print $1}' access.log | sort | uniq -c | sort -nr | head
```

---

### 找 500 错误

```bash
grep " 500 " access.log
```

---

### 找访问最多的 URL

```bash
awk '{print $7}' access.log | sort | uniq -c | sort -nr | head
```

这种练习比任何教程都更有用。
